 1. Justify(выравнивание по ширине) - растягивает текст так, чтобы он занимал всю ширину строки, добавляя пробелы между словами <br>
 2. Spacer (modifier = Modifier.weight(1f)) - 1f (float - число с плавающей точкой), адаптируется под экран, отвечает за распределение пространства, в моем коде их два - в результате контент с фотографией центрирован по вертикали (50% свободного пространства сверху и снизу). <br>
 3. fillMaxWidth(0.5f) - Без 0.5f -заставляет элемент растянуться на всю доступную ширину родителя, с 0.5f - занимает 50% ширины родителя.
